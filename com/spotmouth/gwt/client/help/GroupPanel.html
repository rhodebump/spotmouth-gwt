Group

Name: Give your group a distinctive name.

Description: Give your group a description.

Group Code: You can provide a "Group Code" for you group. Group codes are kind of like a password. Anyone that has the group code can become a member of the group. For example, you can tell members of your team to use group code "team1", and members of your team can join the group.

Group Code Prompt: On the group code entry form, where users would enter the group code, we display this text here.

Visible to members: If this is set to true, all members of the group would be able to see other members of the group.

Manage by members: If this is set to true, all members of the group can manage the group.

Open enrollment. If this is set to true, anybody can join the group.

Use the "Manage Members" to add and remove members of your group.

If you select a member, and click "member details", you can make that member an administrator of your group.
