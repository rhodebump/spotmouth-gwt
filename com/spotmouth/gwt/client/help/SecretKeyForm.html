Secret Key

A secret key is kind of like a password. If you put in a the correct secret key, you would be able to unlock the item that was uses this secret key.

You can use secret keys to create fun geolocation games. For instance, you could have a contest in which you announce that you need to go to the Empire State Building and enter a secret key of "tower". Once there, you enter the secret key, and would be able to view that item.
